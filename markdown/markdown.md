MarkDown笔记
===========

段落和换行
----------

1. 一个以上的空格   
2. 两个以上的空格加换行

标题
----

1. 最高阶标题

    This is an H1   
    =============

2. 第二阶标题

	This is an H2   
	-------------

3. 1到6阶标题

	#This is an H1   
	##This is an H2   
	######This is an H6   

        闭合标题

	#This is an H1   
	###This is an H3###   

引用
----

1. 每行前面加上>   
2. 可嵌套

列表
----

1. 无序列表,每行前加*,+,-   
2. 有序列表,每行前加数字和点

程序代码块
----------

* 缩进 4 个空格或是 1 个制表符

分隔线
------

* 三个或以上的星号、减号

链接
----

1. 方括号后面马上接着圆括号并插入网址链接   
   This is [an example](http://example.com/ "Title") inline link.
    This is [an example](http://example.com/ "Title") inline link.
